# price-compare

Multi shop srovnavac cen pro ceske e-shopy.

## Funkce

- Vyhledavani produktu najednou na vice shopech
- Podporovane shopy: **Alza**, **Datart**, **CZC**, **Bazos**, **Heureka**, **Mall**
- Otevreni vysledku v novych tabech jednim klikem
- Tabulka vysledku s razenim podle shopu, nazvu nebo ceny
- Tlacitko pro vymazani vysledku
- Dark mode design
- Responzivni (funguje i na mobilu)

## Struktura projektu

```
price-compare/
  index.html              # Hlavni stranka
  assets/
    css/
      styles.css          # Styly
    js/
      search.js           # Logika vyhledavani a tabulky
  README.md
```

## Jak to pouzivat

1. Otevri stranku (viz GitHub Pages link nize)
2. Zadej co hledas (napr. `RTX 4060`)
3. Zatrhni shopy, ktere chces prohledat
4. Klikni **Vyhledat** nebo zmackni Enter
5. Kazdy vybrane shop se otevre v novem tabu
6. Vysledky muzes rucne pridat do tabulky
